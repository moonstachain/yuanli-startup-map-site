### A1.9 · 8 个学员的共同特征

把 8 个学员放一起，能看到 **6 个共性**：

```
共 性 1 · 甜用户都是利基市场（FL-CS-02）
        — 不做大众市场，只做"小而美"
        — 8 / 8 都选了利基

共 性 2 · 都进入了高心理账户
        — 至少进入"情绪 / 社交 / 投资"账户
        — 没有一个在做"功能账户"价格战

共 性 3 · 都有强烈的创始人 IP（BL-KR-27）
        — 8 / 8 都把自己 IP 化
        — 用人就是公司，公司就是人

共 性 4 · 都用 K6 主战场原子
        — AI 工具栈 / 私域社群 / 终身会员 / 按效果付费
        — 都在顺势

共 性 5 · 都有"克制自己"的觉悟
        — 知道什么不该做
        — 都过了"卖艺不卖身"的关

共 性 6 · 都有 4 关串行通关的路径
        — 觉醒 → 独创 → 升维 → 锁定
        — 没有跳关，没有偷懒
```

### 🔧 工具 1 · 麦肯锡 30 秒电梯法则

| 字段 | 内容 |
|---|---|
| **归属关** | 第二关 · 入脑表达 |
| **目的** | 30 秒让用户记住且愿意复述 |
| **公式** | 帮谁 + 解决什么 + 怎么做 + 不同点 |
| **步骤** | ① 明确目标用户（不超过 1 类）② 识别核心痛点（最痛的 1 个）③ 陈述解决方案（最独特的 1 个动作）④ 对比同行（你和别人最大的 1 个差别）|
| **经典案例** | 花姐 60 秒模拟客户：体检（功能）+ 靠谱（情绪）+ 背书（社交）+ 合规招投标（避险）四账户全占 |
| **反面教材** | 讲半小时讲不清楚 / PPT 100 页 / 老人小孩听不懂 |
| **AI Prompt** | 基于我的产品描述，生成 3 版麦肯锡 30 秒电梯演讲，每版标注"目标用户 / 痛点 / 解决方案 / 差异化"四要素并打分 |

### 🔧 工具 2 · 用户画像反弹 9 问

| 字段 | 内容 |
|---|---|
| **归属关** | 第二关 · 甜用户 |
| **目的** | 挖掘用户真实任务而非观点 |
| **核心原则** | 不问"你觉得怎么样"——只问"过去发生的具体行为和感受" |
| **9 个问题** | 决策 3 问 + 交付 2 问 + 改变 2 问 + 推荐 2 问 |
| **关键技巧** | 当对方信息有限时，提醒"**举个例子**"——信息量翻 10 倍 |
| **经典案例** | 涛哥访谈高净值家长 → 发现真任务不是考高分而是 AI 时代入场券 |

│                                                    │
└──────────────────────────────────────────────────┘

### A3.2 四关评分锚点（每关 25%）

```
第 一 关 · 原 力 觉 醒（25%）

评分维度：三圈交集完整度
- 高分锚点：三圈同时为真 + 持续 1 年 + 不漂移
- 中分锚点：两圈为真，一圈在补
- 低分锚点：三圈缺失或频繁切换赛道

第 二 关 · 品 类 独 创（25%）

评分维度：甜用户锁定 + 心理账户
- 高分锚点：S 级客户 ≥ 5 + 任务落在社交 / 投资账户
- 中分锚点：S 级客户 2-4 + 任务落在情绪账户
- 低分锚点：客户散乱 + 任务停留在功能账户

第 三 关 · 模 式 升 维（25%）

评分维度：九宫格 + 利润健康
- 高分锚点：9 张测试卡 ≥ 5 张跑通 + 毛利 ≥ 50%
- 中分锚点：3-4 张测试卡跑通 + 毛利 30-50%
- 低分锚点：无系统测试 + 毛利 < 20%

第 四 关 · 壁 垒 锁 定(25%)

评分维度：16 控点 + 30 法则
- 高分锚点：16 控点 ≥ 8 个满分 + 未来十年 > 过去十年
- 中分锚点：16 控点 4-7 个达标 + 部分壁垒在建
- 低分锚点：无壁垒 + 利润逐年下滑
```

### A3.3 总分公式

```
总分 = 第一关 × 25% + 第二关 × 25% + 第三关 × 25% + 第四关 × 25%

每关每个维度 0-100 分

加总后映射到 SABC 段位：
- 总分 85+ → S 级
- 总分 70-84 → A 级
- 总分 55-69 → B 级
- 总分 < 54 → C 级
```

## A5 · MECE 体检 · 8 层结构验证

> **核心命题**：MECE = Mutually Exclusive Collectively Exhaustive = 互斥且穷尽。整本绿皮书的 8 层结构都要过 MECE 体检。

### A5.1 MECE 的两个标准

```
互斥（Exclusive）：
   每个分类的元素之间不重叠
   "天底下的事被你的分类切成不重叠的几块"

穷尽（Exhaustive）：
   所有元素被你的分类覆盖完
   "天底下的事都能放进你的某一块"
```

### A5.2 8 层结构 MECE 体检

```
┌──────────────────────────────────────────────────────────┐
│  层 1 · 顶层公式：财富 = 借势 × 合力                       │
│         互斥：势归外部，力归个人，完全独立 ✓               │
│         穷尽：所有财富归因不外乎天时人和 ✓                  │
│         PASS ✓                                              │
├──────────────────────────────────────────────────────────┤
│  层 2 · 四关闯关：觉醒 → 独创 → 升维 → 锁定                │
│         互斥：输入端 / 市场端 / 变现端 / 防御端 ✓          │
│         穷尽：商业闭环全部生死位 ✓                          │
│         PASS ✓                                              │
├──────────────────────────────────────────────────────────┤
│  层 3 · 第一关·三圈交集：非对称 ∩ 非理性 ∩ 非线性          │
│         互斥：能力（输入）/动机（过程）/结果（输出）✓      │
│         穷尽："投入-产出"闭环全部要素 ✓                    │
│         PASS ✓                                              │
├──────────────────────────────────────────────────────────┤
│  层 4 · 第二关·三角定位：甜用户 × 贵任务 × 入脑            │
│         互斥：Who / What / How 三维独立 ✓                  │
│         穷尽：用户认知完整链路 ✓                            │
│         PASS ✓                                              │
├──────────────────────────────────────────────────────────┤
│  层 5 · 四大心理账户：功能 / 情绪 / 社交 / 投资            │
│         互斥：四账户预算池完全独立 ✓                        │
│         穷尽：覆盖人的全部支付动机 ✓                        │
│         PASS ✓                                              │
├──────────────────────────────────────────────────────────┤
│  层 6 · 第三关·三链贯通：前 / 后 / 财                      │
│         互斥：渴望度 / 可行性 / 生存性独立 ✓               │
│         穷尽：完整商业循环 ✓                                │
│         PASS ✓                                              │
├──────────────────────────────────────────────────────────┤
│  层 7 · 第四关·四维护城：虚 × 实 × 入 × 出                 │
│         互斥：供给侧 vs 需求侧独立 ✓                        │
│         穷尽：商业防御全维度 ✓                              │
│         PASS ✓                                              │
├──────────────────────────────────────────────────────────┤
│  层 8 · 加速器层：AI 杠杆 + 时间复利                       │
│         互斥：空间维度 + 时间维度 ✓                        │
│         穷尽：覆盖所有加速可能 ✓                            │
│         PASS · 不污染顶层公式 ✓                            │
└──────────────────────────────────────────────────────────┘

★ 8 层结构 MECE 体检 PASS · 体系完整无裂缝
```

### A5.3 MECE 的运用

```
□ 你做的任何分类，自问：
  ① 我的分类是否互斥（元素之间没重叠）？
  ② 我的分类是否穷尽（覆盖所有元素）？

□ 你的商业模式九宫格，自问：
  ① 9 个模块之间互斥吗？
  ② 9 个模块加起来穷尽商业循环吗？

□ 你的产品阶梯，自问：
  ① 5 个价位互斥吗（199 / 9980 / 29800 / 99800 / 168000）？
  ② 加起来穷尽用户需求层级吗？
```

**MECE 不是奢侈品——是商业思维的基本素养**。

## A7 · 术语表 · 35+ 核心概念

> **核心命题**：35+ 个概念的完整定义，方便快速回查。

### 顶层公式

- **借势**：顺应时代趋势和经济周期，吃到时代红利。度量：康波周期 K1-K6
- **合力**：向内求，找到能力原点（原力）。度量：12 原型人格
- **原力**：你的能量原点 = 非对称优势 ∩ 非理性痴迷 ∩ 非线性回报
- **康波周期**：尼古拉·康德拉季耶夫提出的 60 年长波周期，迄今五次已发生（K1-K5），K6 = AI / 生物科技
- **12 原型人格**：荣格分析心理学：英雄/反叛者/魔法师/统治者/关怀者/创造者/情人/小丑/普通人/天真者/智者/探索者

### 第一关 · 原力觉醒

- **非对称优势**：区别于卷学历卷身高的对称竞争，是你独有的、别人无法模仿的特质
- **非理性痴迷**：即使没有回报甚至倒贴钱也要做的事，是你的天命（Calling）
- **非线性回报**：做这件事扩 100 倍不需要线性增加精力，越做越省力
- **天才区**：做得好且充电、越做越精神的能力领域；区别于"卓越区"（做得好但耗能）
- **阴影即力量**：荣格概念：被压抑的负面特质反面就是超能力。修法：3-2-1 阴影练习
- **金色阴影**：通过阴影整合，将负面特质转化为正面资产

### 第二关 · 品类独创

- **甜用户**：你最强禀赋能服务、且这辈子愿意为你买单的核心人群
- **贵任务**：用户雇佣你的产品去完成的高价值任务（Jobs-to-be-Done）
- **心理账户**：塞勒诺贝尔奖理论：人对不同用途的钱有完全独立的预算池。本体系四账户：功能 / 情绪 / 社交 / 投资
- **一句入脑**：麦肯锡 30 秒电梯法则：帮谁 + 解决什么 + 怎么做 + 不同点
- **品类独创**：16 字真言：无法分类 / 重新定义 / 自带预算 / 品类独创

### 第三关 · 模式升维

- **前链路**：渴望度架构：用户细分 + 价值主张 + 渠道结构 + 客户关系
- **后链路**：可行性架构：核心资源 + 关键业务 + 伙伴网络
- **财链路**：生存性架构：收入来源 + 定价机制 + 成本结构
- **九宫格**：商业模式画布 10 模块的简化表述（3 链 × 3-4 模块）
- **58 原子**：商业模式 58 种 DNA 组件，覆盖 300 年商业史
- **闭环验证测试卡**：MVP 级模式测试：2 万元 / 4 周 / 10 倍空间筛选标准

### 第四关 · 壁垒锁定

- **壁垒锁定**：四维护城：虚（品牌）+ 实（规模）+ 入（锁定）+ 出（网络）
- **16 战略控制点**：巴菲特/段永平财富审美：4 维 × 4 控点 = 16 个壁垒评分位
- **30 法则**：段永平投资标准：好商业模式 = 未来十年赚的钱 > 过去十年
- **10/10 法则**：伟大企业标准：未来十年 >> 过去十年（≥ 3 倍）
- **数据飞轮**：K6 时代新增第 16 控点——用户越多 → AI 越聪明 → 下一个用户越友好
- **病毒系数**：每个用户能带来多少新用户。≥ 1 时自传播

### 加速器 / 工具

- **AI 杠杆**：AI 工具栈 + 各关 AI 教练
- **时间复利**：每关产物都是可累积资产
- **十万力**：AI 时代核心能力——使唤 AI 的能力
- **Rose 提示词法**：AI 提示词 4 要素：Role + Objective + Strategy + Example
- **游轮测试**：创始人消失 60 天，团队和系统能否自运转

### 诊断 / 评分

- **SABC 能量级**：企业诊断分级：S 穿越周期 / A 模式跑通 / B 在跑测试 / C 还在试错
- **ABCS 段位**：用户表达成熟度：S 模型级 / A 付费级 / B 叫好级 / C 想法级
- **MECE**：Mutually Exclusive Collectively Exhaustive：互斥且穷尽，麦肯锡核心方法论
- **第一性原理**：回到最基础的、不可再分解的真理出发推导

### 其他概念

- **骑象人 vs 大象**：骑象人 = 理性脑（1 倍能量）；大象 = 情感脑（1000 倍能量）
- **压强原理**：缩小目标面积比加大力气更有效
- **梅特卡夫定律**：网络价值 ∝ 用户数²
- **千日窗口**：康波转换的前 1000 天 = 最佳入场时机（K6 = 2025-2028）
